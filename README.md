# **Mig Cupcakes 🧁**
Este projeto é uma landing page interativa desenvolvida desde a sua prototipagem no **Figma** para a marca fictícia Mig Cupcakes, com foco em transmitir a identidade visual acolhedora e sofisticada de uma confeitaria artesanal.

## **✨ Funcionalidades**
**Layout responsivo** com tipografia elegante e cores quentes que remetem ao universo da confeitaria.

**Animações suaves** de transição de cores no fundo e na área de destaque da imagem.

**Troca automática de imagens** dos cupcakes com efeito fade, criando dinamismo na apresentação.

**Loop infinito** de variações visuais, mantendo a página sempre atrativa.

**Integração com ícones de redes sociais**, reforçando a presença digital da marca.

## **🛠️ Tecnologias utilizadas**
**HTML5** para estruturação semântica.

**CSS3** para estilização, responsividade e transições animadas.

**JavaScript** para manipulação dinâmica de cores e imagens.

**Google Fonts** para tipografia personalizada.

**Tabler Icons** para uma interatividade mais intuitiva.

## **🚀 Objetivo**
O projeto foi criado como exercício de **design front-end** e **animação com JavaScript**, simulando uma página de apresentação de uma confeitaria. Ele demonstra como combinar estética, interatividade e código limpo para entregar uma experiência agradável ao usuário.

# **Mig Cupcakes 🧁**
This project is an interactive landing page developed from its initial Figma prototyping for the fictional brand Mig Cupcakes. It focuses on conveying the warm and sophisticated visual identity of a handcrafted bakery.

## **✨ Features**
**Responsive Layout** featuring elegant typography and warm colors that evoke the confectionery world.

**Smooth Animations** with color transitions in the background and the image highlight area.

**Automatic Image Swapping** of cupcakes with a fade effect, creating a dynamic presentation.

**Infinite Loop** of visual variations to keep the page consistently engaging.

**Social Media Integration with icons** to strengthen the brand's digital presence.

## **🛠️ Technologies Used**
**HTML5** for semantic structuring.

**CSS3** for styling, responsiveness, and animated transitions.

**JavaScript** for dynamic manipulation of colors and images.

**Google Fonts** for custom typography.

**Tabler Icons** for a more intuitive interactivity.

## **🚀 Objective**
This project was created as a **front-end design** and **JavaScript animation exercise**, simulating a bakery's presentation page. It demonstrates how to combine aesthetics, interactivity, and clean code to deliver an enjoyable user experience.
